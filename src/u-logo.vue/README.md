# Logo

## 示例
### 基本形式

``` html
<u-linear-layout direction="vertical">
    <div style="background: #4289db; padding: 0 10px;">
        <u-logo></u-logo>
    </div>
    <div style="background: #1d2b40; padding: 0 10px;">
        <u-logo href="http://www.baidu.com" target="_blank" name="163yun"></u-logo>
    </div>
</u-linear-layout>
```
